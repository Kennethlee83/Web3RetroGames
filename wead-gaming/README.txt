WeAD Gaming Website Setup
========================

1. Download XAMPP from https://www.apachefriends.org/
2. Install XAMPP
3. Copy this folder to: C:\xampp\htdocs\
4. Start Apache in XAMPP Control Panel
5. Visit: http://localhost/wead-gaming/

Links to update:
- Telegram: Already set to https://t.me/WeAdvertizers
- Discord: Replace "#" with your Discord invite link

That's it!
